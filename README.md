# 84-Key Mechanical Keyboard (Hack Club Blueprint)

This is my custom 84-key mechanical keyboard PCB project, designed from scratch for the Hack Club **Blueprint** program.

##  Features
- 84-key compact layout (similar to a 75%)
- MX-style switches
- Plate-mounted stabilizers
- Designed in KiCad
- Powered by Pro Micro (ATmega32U4)
- QMK-compatible firmware
